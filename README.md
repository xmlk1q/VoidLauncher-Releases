# VoidLauncher

**VoidLauncher** is a premium, ad-free desktop game patcher for Windows. It gives you a clean, modern interface to detect your installed games, apply official Void Nexus patches, and launch — without ads, bloatware, or third-party installers.

---

## Download

**Get the latest installer from the [Releases](https://github.com/xmlk1q/VoidLauncher-Releases/releases) page.**

1. Open this repository on GitHub.
2. Click **Releases** in the right-hand sidebar (or use the link above).
3. Download the latest `VoidLauncher-Setup-x.x.x.exe`.
4. Run the installer and follow the on-screen steps.

> This repository contains release artifacts only. The application source code is maintained in a separate private repository.

---

## What VoidLauncher Does

- **Detects your game install** automatically via Steam library paths, with a manual folder picker as a fallback
- **Injects bypass patches** securely through the Void Nexus API — streamed, extracted, and applied in place
- **Arabic localization** — one-click Arabize patch for supported titles
- **Launches your game** directly from the launcher
- **Auto-updates** — the installed app checks this repository for new versions and notifies you when an update is available

## System Requirements

- **OS:** Windows 10 or Windows 11 (64-bit)
- **Disk space:** ~100 MB for the launcher; additional space for game patches
- A legitimate copy of the supported game installed via Steam

---

## Installation

1. Download `VoidLauncher-Setup-x.x.x.exe` from **Releases**.
2. Run the installer (you may change the install directory).
3. A desktop shortcut and Start Menu entry will be created automatically.
4. Open VoidLauncher, locate your game if it is not detected automatically, and use the action buttons on the game detail page.

---

## Updating

VoidLauncher checks for updates automatically when you run the installed application. When a new version is available, you will be prompted to download and install it.

You can also visit the [Releases](https://github.com/xmlk1q/VoidLauncher-Releases/releases) page at any time to download the latest installer manually.
